nocnoc gives marketplace sellers a single REST API to publish products, handle orders, and manage shipments — all from `live.nocnocstore.com`. Whether you're building an integration from scratch or automating an existing catalog, this documentation covers everything you need.

## What you can do

<CardGroup cols={2}>
  <Card title="Publish products" icon="package">
    List your catalog, update stock and prices, and retrieve product data by SKU. Create listings manually or from an Amazon ASIN.
  </Card>
  <Card title="Manage orders" icon="shopping-cart">
    Fetch incoming orders with filters for status and date range. Cancel orders when needed.
  </Card>
  <Card title="Ship packages" icon="truck">
    Register individual Son packages per order, then consolidate them into Father packages for first-mile shipping.
  </Card>
  <Card title="Sell across LATAM" icon="globe">
    One integration connects your catalog to the leading marketplaces across Latin America.
  </Card>
</CardGroup>

## Base URL

All requests go to the live environment:

```
https://live.nocnocstore.com/api/v2
```
