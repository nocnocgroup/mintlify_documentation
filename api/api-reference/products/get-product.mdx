---
title: "Get product by SKU"
api: "GET https://live.nocnocstore.com/api/v2/products/sku/{sku}"
description: "Returns the complete information for a single product identified by its SKU."
---

## Path parameters

<ParamField path="sku" type="string" required>
  The unique SKU of the product.
</ParamField>

## Example request

```bash
curl https://live.nocnocstore.com/api/v2/products/sku/MY-SKU-001 \
  -H "X-Api-Key: YOUR_SELLER_KEY"
```

## Responses

<ResponseField name="200" type="object">
  Returns the complete product data for the given SKU.
</ResponseField>

<ResponseField name="401" type="object">
  Missing or invalid API key.
</ResponseField>

<ResponseField name="404" type="object">
  SKU not found.
</ResponseField>
