---
title: "Create Son package"
api: "POST https://live.nocnocstore.com/api/v2/packages"
description: "Creates a Son package for a single order. Each order requires its own Son package."
---

<Note>
  nocnoc manages two types of packages: **Son** and **Father**. Create a Son package per order, then group multiple Son packages into a [Father package](/api/api-reference/packages/create-father-package) for consolidated first-mile shipping.
</Note>

## Body parameters

<ParamField body="order_reference_id" type="string" required>
  The reference ID of the order included in this package.
</ParamField>

<ParamField body="labels" type="array" required>
  Array of label objects containing first-mile tracking and nocnoc labels. Minimum 1, maximum 10.
</ParamField>

<ParamField body="items" type="array" required>
  Array of item objects inside the package. Minimum 1, maximum 10.
</ParamField>

## Returns

The created Son package object with its package ID, which you'll need to create a Father package.

## Example request

```bash
curl -X POST https://live.nocnocstore.com/api/v2/packages \
  -H "X-Api-Key: YOUR_SELLER_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "order_reference_id": "REF-123456",
    "labels": [
      {
        "type": "first_mile",
        "url": "https://example.com/label.pdf"
      }
    ],
    "items": [
      {
        "sku": "MY-SKU-001",
        "quantity": 1
      }
    ]
  }'
```

<ResponseExample>
```json 201
{
  "package_id": "PKG-SON-789",
  "order_reference_id": "REF-123456",
  "type": "son",
  "status": "created"
}
```
</ResponseExample>
